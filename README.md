# 🎓 AI-Based Dropout Prediction and Counseling System

An intelligent web-based application that uses **Machine Learning** to identify students who are at risk of dropping out and provides personalized counseling recommendations. The system helps educational institutions monitor student performance, predict dropout risk at an early stage, and support timely interventions to improve student retention.

---

## ✨ Features

- 👨‍🎓 Student Registration
- 🔐 Secure Login System
- 📊 Interactive Dashboard
- 🤖 AI-Based Dropout Prediction
- 💡 Personalized Counseling Recommendations
- 📑 Student Reports and Analytics
- 🗄️ SQLite Database Integration
- 🌲 Random Forest Machine Learning Model
- 📈 Data Visualization
- 📝 Counselor Notes Management

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Based-Dropout-Prediction-System.git

cd AI-Based-Dropout-Prediction-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```

The application will start locally and can be accessed through your web browser.

---

## 🤖 Machine Learning Model

The system uses a **Random Forest Classifier** to predict student dropout risk based on academic, attendance, financial, and behavioral data. The trained model classifies students into different risk levels and generates explainable counseling recommendations to support early intervention.

---

## 🗄️ Database

The application uses **SQLite** as its database to store:

- Student Information
- User Login Details
- Prediction Results
- Counseling Records
- Student Reports
